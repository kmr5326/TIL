### CNN vs RNN
CNN: 공간적 특징(이미지 등)을 추출하는 데 특화

RNN: 순서가 있는 데이터(시계열, 텍스트)에 특화
### BatchNorm / Dropout
BatchNorm: 각 층의 입력 분포를 정규화해 학습을 안정화하고 속도를 향상시키는 기법

Dropout: 학습 과정에서 일부 뉴런을 무작위로 제거해 과적합을 방지하는 기법

### LLM 기본 구조
1. 텍스트를 토큰화한다.
2. 토큰을 임베딩하여 벡터로 변환
3. Transformer 구조의 여러 Self-attention 블록을 통과하여 확률 계산
4. 가장 확률 높은 단어 생성

### RAG
RAG는 외부 시스템에서 관련 정보를 검색한 뒤 이를 기반으로 LLM이 답변을 생성하는 구조
